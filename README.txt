В проекте производится расчет заработной платы (бонуса)
брокера (по разным вариантам), в зависимости от стоимости
сделки, по прогрессивной шкале.

Применен паттерн MVC (Model-View-Controller).
Model - содержит бизнес-логику (расчеты, работа с БД и т.п.).
View - ввод данных и отображение результата манипуляций над
ними. Презентация.
Controller - посредник между Model и View.
Разделяет бизнес-логику и презентацию. Манипулирует движением данных.
Назначает исполнителя на обработку данных.
